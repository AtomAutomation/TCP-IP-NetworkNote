## 1.12 I/O复用

并发服务器的第二种实现方法，基于I/O复用的服务器端构建

### 12.1 基于I/O复用的服务器端

#### 12.1.1 多进程服务器端的缺点和解决办法

创建进程时需要付出很大代价，需要大量的运算和内存空间。由于每个进程都具有独立的内存空间，所以相互间的数据交换也要求采用相对复杂的方法。

#### 12.1.2 理解复用

复用: 为了提供物理设备的效率，用最少的物理要素传递最多的数据时使用的技术
